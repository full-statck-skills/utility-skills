## 📖 简介

**技能管理工具** 是一组 AI 编码智能体技能，属于 [Full Stack Skills](https://github.com/partme-ai/full-stack-skills) 生态，由 [PartMe.AI](https://github.com/partme-ai) 维护。包含 **3 个技能**。

## 📦 安装

```bash
npx skills add full-statck-skills/utility-skills
```
